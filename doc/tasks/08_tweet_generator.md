# 08: ツイート生成ロジックの実装

## ゴール

ユーザーデータとテンプレートを組み合わせて、ランダムにツイートを生成するエンジンを実装する。

## 成果物

- `js/generator.js` にツイート生成ロジックを実装

## 完了条件

- ランダムにユーザーを選択できる
- ユーザーの性格タイプに応じた重み付けでカテゴリを選択できる（例: オタク系ユーザーはオタク系カテゴリが出やすい）
- テンプレートのプレースホルダーが語彙データで正しく埋められる
- リプライ風ツイートで、実在する別ユーザーの@スクリーンネームが使われる
- 生成されるツイートに十分なバリエーションがあり、短時間では同じ内容が繰り返されにくい
- `generateTweet()` のような関数を呼ぶと、1件のツイートオブジェクト（ユーザー情報、本文、タイムスタンプ含む）が返る
