# 01: プロジェクトセットアップ & HTML骨格

## ゴール

プロジェクトのディレクトリ構成を作成し、index.htmlに旧Twitter風3カラムレイアウトのHTML骨格を構築する。ブラウザで開いて構造が確認できる状態にする。

## 成果物

- `index.html` — 3カラムレイアウトのセマンティックHTML
- `css/style.css` — 空ファイル（リンク済み）
- `js/app.js`, `js/users.js`, `js/tweets.js`, `js/generator.js` — 空ファイル（リンク済み）

## 完了条件

- ブラウザでindex.htmlを開くと、ヘッダー・左サイドバー・タイムライン・右サイドバーの4領域が確認できる
- 各JS/CSSファイルが正しくリンクされている
- HTMLにダミーテキストが入っており、レイアウト構造が視認できる
