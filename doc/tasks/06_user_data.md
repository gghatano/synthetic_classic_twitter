# 06: 架空ユーザーデータの定義

## ゴール

15〜20人の架空ユーザーデータを定義する。各ユーザーは表示名、スクリーンネーム、アイコン色、性格タイプを持つ。

## 成果物

- `js/users.js` に架空ユーザーデータを実装

## 完了条件

- 15〜20人のユーザーが定義されている
- 各ユーザーが以下の属性を持つ:
  - 表示名（日本語、例: たかし、みく）
  - スクリーンネーム（例: @takashi_2010）
  - アイコン色（CSS色コード）
  - 性格タイプ（日常系 / ネタ系 / 意識高い系 / オタク系 など）
- 性格タイプにバリエーションがあり、タイムラインが多様になるよう配分されている
