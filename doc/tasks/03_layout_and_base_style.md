# 03: 3カラムレイアウト & ベーススタイル

## ゴール

ページ全体の3カラムレイアウトと、旧Twitterのカラーパレット・フォント・背景などのベーススタイルを実装する。

## 成果物

- `css/style.css` にレイアウト・ベーススタイルを追加

## 完了条件

- 背景色が旧Twitter風（#E8F0FE〜#F5F5F5系）
- 3カラム（左サイドバー / タイムライン / 右サイドバー）が適切な幅比率で表示される
- フォントが `"Helvetica Neue", Arial, "Hiragino Kaku Gothic Pro", Meiryo, sans-serif` に設定されている
- ツイート領域の背景が白、境界線が#E8E8E8で旧Twitter風に見える
