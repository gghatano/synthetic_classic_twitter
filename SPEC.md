# Old Twitter - 仕様書

## 概要

2010年前後の旧Twitterの見た目・雰囲気を再現した静的Webアプリ。
架空のユーザーたちが当時流行した投稿スタイルで次々とツイートを流していく、眺めて楽しむタイムラインビューア。

GitHub Pagesでホストする。

---

## コンセプト

- 懐かしの旧Twitter UIを忠実に再現する（2010〜2012年頃のデザイン）
- 当時のTwitter文化（なう、わず、だん等）を反映したツイートが自動生成される
- ユーザーは眺めるだけ。操作不要で次々とツイートが流れてくる

---

## 技術構成

| 項目 | 選定 |
|------|------|
| ホスティング | GitHub Pages |
| フレームワーク | なし（Vanilla HTML/CSS/JS） |
| ビルドツール | なし（静的ファイルのみ） |
| 外部API | なし（すべてクライアントサイドで完結） |

### 理由

- GitHub Pagesの制約上、静的ファイルのみ
- シンプルな要件のためフレームワーク不要
- 外部依存なしで長期間メンテナンスフリー

## 画面構成

### レイアウト（3カラム）

旧Twitterの3カラムレイアウトを再現する。

```
+-----------------------------------------------------+
|  ヘッダーバー（水色背景、Twitterロゴ、検索窓風）       |
+------------+--------------------+-------------------+
|            |                    |                   |
| 左サイド   |   タイムライン      |  右サイド          |
| バー       |   （メイン）        |  バー             |
|            |                    |                   |
| ・プロフ   |  ツイートが次々と   |  ・トレンド       |
|  ィール    |  流れてくる         |  ・おすすめ       |
|  カード    |                    |    ユーザー       |
| ・ツイート |                    |                   |
|  入力欄風  |                    |                   |
|            |                    |                   |
+------------+--------------------+-------------------+
```

### ヘッダーバー

- 旧Twitterの水色（#33CCFF → 後期は #55ACEE に近い色）のグラデーション背景
- 左側: Twitter旧ロゴ風テキスト（「Twitter」のロゴ文字、または鳥アイコン風の文字）
- 中央: 検索バー風のダミー入力欄
- 右側: ダミーのナビゲーションリンク（ホーム、@つながり、ダイレクトメッセージ風）

### 左サイドバー

- ログインユーザー風のプロフィールカード
  - アイコン（デフォルトたまごアイコン風）
  - 名前: 適当な日本語名
  - ツイート数 / フォロー数 / フォロワー数（ダミー）
- 「いまどうしてる？」のツイート入力欄（装飾のみ、入力不可）

### メインタイムライン

- ツイートが一定間隔で上部に追加されていく
- 各ツイートの構成:
  - ユーザーアイコン（カラフルなデフォルトアイコン）
  - ユーザー名 + @スクリーンネーム
  - ツイート本文
  - タイムスタンプ（「○秒前」「○分前」形式）
  - 「返信」「リツイート」「お気に入り（星マーク）」のダミーアイコン
- 新しいツイートは上から挿入され、自然に下にスクロールされていく
- 最大表示件数は50件（超えたら下から削除）

### 右サイドバー

- 「トレンド」セクション（ダミーのハッシュタグ一覧）
- 「おすすめユーザー」セクション（ダミー）

## ツイート自動生成システム

### 生成間隔

- 1〜5秒のランダム間隔で新しいツイートを生成・表示
- たまに連続投稿（0.3秒間隔で2〜3件）も発生させる

### 架空ユーザー

15〜20人程度の架空ユーザーを事前定義する。

各ユーザーは以下の属性を持つ:

| 属性 | 例 |
|------|----|
| 表示名 | たかし、みく、けんじ など |
| スクリーンネーム | @takashi_2010、@miku_love など |
| アイコン色 | CSSで生成（#FF6B6B等） |
| 性格タイプ | 日常系 / ネタ系 / 意識高い系 / オタク系 など |

### ツイート内容テンプレート

当時のTwitter文化を反映した投稿パターンを用意する。

#### カテゴリ1: 「なう」系

- `{場所}なう`
- `{食べ物}なう`
- `{活動}なう`
- 例: 「渋谷なう」「ラーメンなう」「バイトなう」

#### カテゴリ2: 「わず」系（完了報告）

- `{活動}わず`
- `{場所}から帰宅わず`
- 例: 「部活わず」「バイトわず」「テストわず」

#### カテゴリ3: 「だん」系（進行中）

- `{活動}だん`
- 例: 「移動だん」「勉強だん」

#### カテゴリ4: 日常ツイート

- `腹減った`
- `眠い…`
- `暇だー`
- `{天気}だなー`
- `今日も一日がんばるぞい`
- `おはようございます`
- `おやすみなさい`

#### カテゴリ5: テンション系

- `ｗｗｗｗｗｗ`
- `やばいｗｗｗ`
- `まじかよｗｗｗ`
- `神ｗｗｗ`
- `クソワロタｗｗｗ`
- `ちょｗｗｗ`

#### カテゴリ6: 意識高い系

- `今日の学び: {格言風テキスト}`
- `感謝。ただそれだけ。`
- `人生は選択の連続`
- `行動あるのみ`

#### カテゴリ7: オタク系

- `{アニメ名}{話数}話見た。{感想}`
- `{キャラ名}まじ天使`
- `新刊買ってきた`
- `コミケの準備しなきゃ`

#### カテゴリ8: リプライ風

- `@{他ユーザー} {返信内容}`
- `@{他ユーザー} ｗｗｗ`
- `@{他ユーザー} それなｗ`

#### カテゴリ9: 定番フレーズ

- `誰か構ってー`
- `リムられた…`
- `フォロバありがとうございます！`
- `TL追えない`
- `ついったー楽しい`
- `寝落ちしてた`

#### カテゴリ10: ハッシュタグ系

- `#○○好きな人RT`
- `#nowplaying {曲名} - {アーティスト}`
- `#followme`

### ツイート生成ロジック

1. ランダムにユーザーを選択
2. そのユーザーの性格タイプに応じた重み付けでカテゴリを選択
3. カテゴリ内のテンプレートをランダム選択
4. プレースホルダーをランダムな語彙で埋める
5. タイムスタンプを生成（「○秒前」等）

## 動作仕様

### 初期表示

1. ページ読み込み時、タイムラインに初期ツイートを10件表示
2. 1秒後から自動生成開始

### タイムライン更新

1. 1〜5秒のランダム間隔で新ツイートを生成
2. タイムライン上部にアニメーション付きで挿入
3. 50件を超えたら最下部のツイートを削除

### タイムスタンプ

- 表示時は「たった今」
- 時間経過に応じて「○秒前」→「○分前」に更新
- 1分ごとに全ツイートのタイムスタンプを更新

### レスポンシブ対応

- デスクトップ: 3カラム表示
- タブレット（768px以下）: サイドバーを非表示、タイムラインのみ
- モバイル（480px以下）: コンパクトレイアウト
